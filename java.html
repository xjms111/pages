Java

Java to współbieżny, oparty na klasach, obiektowy język programowania ogólnego zastosowania. Został stworzony przez grupę roboczą pod kierunkiem Jamesa Goslinga z firmy Sun Microsystems.

Jego podstawowe koncepcje zostały przejęte z języka Smalltalk (maszyna wirtualna, zarządzanie pamięcią) oraz z języka C++ (duża część składni i słów kluczowych).

Moja historia z Javą

W Javie zacząłem programować na początku swoich studiów informatycznych, w semestrze zimowym 2024 i to właśnie w niej wykonałem największą liczbę projektów.
